Atividade 4: Parâmetros/atributos para o <input>

Por Samuel Carvalho

Autocomplete:

Permite ao browser prever o valor a ser digitado com base em valores anteriores. Pode ser "setado" on (ligado, padrão) ou off (desligado)



Autofocus:

Torna o campo de input automaticamente focalizado quando a página carrega.


Disabled:

Desativa o campo de input.


Max e min:

Determina o máximo e o mínimo de valores que podem ser aceitos no campo de input.
Funciona com os seguintes tipos de valores (types): number, range, date, datetime-local, month, time, week.


Maxlength:

Determina o número máximo de caracteres permitidos no campo de input.

Name:

Determina o nome do elemento input.
É usado para referenciar esse elemento em um javascript ou referenciar o dado do formulário depois de este ter sido enviado.

Pattern:

Determina um padrão de valor ao campo de input a ser comparado com o valor digitado e enviado pelo usuário.
Se o valor digitado não corresponder ao padrão especificado, o usuário não poderá prosseguir com o envio do formulário.



Placeholder:

Mostra um exemplo de valor no campo de input.



Readonly

Torna o campo de input disponível apenas para leitura. O usuário não pode modificá-lo, mas pode selecionar e copiar o texto do formulário.


Required:

Determina que para enviar o formulário é necessário preencher aquele campo.

Size:

Especifica a largura visível, em caracteres, de um campo de input.

Type:

Determina o tipo de entrada (input) a ser utilizada.
Dentre os tipos de entrada temos: button (botão), checkbox (caixa de marcação), color (selecionar uma cor), date (data), datetime-local (seleciona uma data e horário sem considerar fuso horário), email, file (arquivo), hidden (um campo invisível ao usuário, porém visível e editável usando dev tools do navegador), image (imagem), month (mês), number, password, radio (selecionar uma dentre a opções marcando a bolinha), range (seleciona um valor usando um slider), reset (retorna os valores do formulário aos valores iniciais), search (funciona como um "text" mas usado para pesquisas), submit (envia o formulário), tel (telefone), time (horário sem fuso horário), url (endereço URL), week (seleciona semana e ano).

Value:

Determina o valor do campo. Pode ser editado se não houver o atributo readonly
